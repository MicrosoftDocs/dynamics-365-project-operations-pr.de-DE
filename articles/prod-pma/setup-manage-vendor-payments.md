# <a name="set-up-and-use-pay-when-paid-vendor-payments"></a><span data-ttu-id="4f453-103">Pay-When-Paid-Kreditorenzahlungen einrichten und verwenden</span><span class="sxs-lookup"><span data-stu-id="4f453-103">Set up and use pay-when-paid vendor payments</span></span>

[!include [banner](../includes/banner.md)]

<span data-ttu-id="4f453-104">Wenn Sie einem Kreditor die Genehmigung erteilen, als Zulieferer zu arbeiten, möchten Sie möglicherweise die Zahlung an den Kreditor zurückhalten, bis Ihr Kunde Sie für das Projekt bezahlt.</span><span class="sxs-lookup"><span data-stu-id="4f453-104">When you approve a vendor to work as a subcontractor, you might want to withhold payment to the vendor until your customer pays you for the project.</span></span> <span data-ttu-id="4f453-105">Um dieses Szenario zu unterstützen, können Sie PWP-Bedingungen (Pay-When-Paid) einrichten, wenn Sie die Bestellung (PO) beim Lieferanten einrichten.</span><span class="sxs-lookup"><span data-stu-id="4f453-105">To support this scenario, you can set up pay-when-paid (PWP) terms when you set up the purchase order (PO) with the vendor.</span></span>

<span data-ttu-id="4f453-106">Wenn ein Kunde beispielsweise einen Betrag auf einer Projektrechnung bezahlt, können Sie einen Teil oder den gesamten Rechnungsbetrag des Kreditors freigeben.</span><span class="sxs-lookup"><span data-stu-id="4f453-106">For example, when a customer pays an amount on a project invoice, you can release some or all of the vendor invoice amount.</span></span> <span data-ttu-id="4f453-107">Richten Sie einfach PWP-Bedingungen ein, die festlegen, dass der Kreditor bezahlt wird, nachdem Sie einen Prozentsatz der entsprechenden Zahlung vom Kunden erhalten haben.</span><span class="sxs-lookup"><span data-stu-id="4f453-107">Just set up PWP terms that specify that the vendor will be paid after you receive a percentage of the related payment from the customer.</span></span> <span data-ttu-id="4f453-108">Wenn Sie eine Teilzahlung von einem Kunden erhalten, können Sie einige der zugehörigen Kreditorenrechnungen manuell zur Zahlung freigeben.</span><span class="sxs-lookup"><span data-stu-id="4f453-108">If you receive partial payment from a customer, you can manually release some of the related vendor invoices for payment.</span></span>

<span data-ttu-id="4f453-109">Das folgende Beispiel beschreibt den Prozess bei Verwendung von PWP-Bedingungen.</span><span class="sxs-lookup"><span data-stu-id="4f453-109">The following example outlines the process when PWP terms are used.</span></span>

## <a name="example"></a><span data-ttu-id="4f453-110">Beispiel</span><span class="sxs-lookup"><span data-stu-id="4f453-110">Example</span></span>

<span data-ttu-id="4f453-111">Ihre Organisation erklärt sich damit einverstanden, einem Kunden 100 Computer mit installierter Software zum Preis von 150,00 US-Dollar (USD) pro Computer zur Verfügung zu stellen.</span><span class="sxs-lookup"><span data-stu-id="4f453-111">Your organization agrees to provide a customer with 100 computers that have software installed, for a price of 150.00 US dollars (USD) per computer.</span></span> <span data-ttu-id="4f453-112">Anschließend beauftragen Sie einen Kreditor mit der Bereitstellung der Computer, auf denen Software installiert ist.</span><span class="sxs-lookup"><span data-stu-id="4f453-112">You then hire a vendor to provide you with the computers that have software installed.</span></span> <span data-ttu-id="4f453-113">Gemäß der Vereinbarung muss der Kunde die Qualität der Computer genehmigen, bevor er Ihre Organisation bezahlt.</span><span class="sxs-lookup"><span data-stu-id="4f453-113">According to the agreement, the customer must approve the quality of the computers before it pays your organization.</span></span> <span data-ttu-id="4f453-114">Die Richtlinie Ihres Unternehmens besteht darin, Zahlungen an Kreditoren zurückzuhalten, bis der Kunde Sie bezahlt hat.</span><span class="sxs-lookup"><span data-stu-id="4f453-114">Your organization's policy is to withhold payment to vendors until the customer has paid you.</span></span> <span data-ttu-id="4f453-115">Daher richten Sie das Projekt so ein, dass es einen PWP-Prozentsatz von 100 Prozent aufweist.</span><span class="sxs-lookup"><span data-stu-id="4f453-115">Therefore, you set up the project so that it has a PWP percentage of 100 percent.</span></span>

<span data-ttu-id="4f453-116">Der Kreditor sendet Ihnen die 100 Computer, auf denen Software installiert ist, zusammen mit einer Rechnung für 10.000,00 USD.</span><span class="sxs-lookup"><span data-stu-id="4f453-116">The vendor sends you the 100 computers that have software installed, together with an invoice for 10,000.00 USD.</span></span> <span data-ttu-id="4f453-117">Da für Ihr Projekt PWP-Bedingungen eingerichtet sind, bezahlen Sie den Kreditor nicht nach Erhalt der Computer.</span><span class="sxs-lookup"><span data-stu-id="4f453-117">Because PWP terms are set up for your project, you don't pay the vendor upon receipt of the computers.</span></span> <span data-ttu-id="4f453-118">Stattdessen senden Sie die Computer zusammen mit einer Rechnung für 15.000,00 an den Kunden.</span><span class="sxs-lookup"><span data-stu-id="4f453-118">Instead, you send the computers to the customer, together with an invoice for 15,000.00.</span></span> <span data-ttu-id="4f453-119">Der Kunde inspiziert die Computer und genehmigt den vollen Betrag der Projektrechnung.</span><span class="sxs-lookup"><span data-stu-id="4f453-119">The customer inspects the computers and approves the full amount of the project invoice.</span></span>

<span data-ttu-id="4f453-120">Nachdem Sie die vollständige Zahlung vom Kunden erhalten haben, zahlen Sie dem Kreditor 10.000,00. Dies ist der volle Betrag der Kreditorenrechnung.</span><span class="sxs-lookup"><span data-stu-id="4f453-120">After you receive the full payment from the customer, you pay the vendor 10,000.00, the full amount of the vendor invoice.</span></span>

## <a name="set-up-pwp-terms-for-a-project"></a><span data-ttu-id="4f453-121">PWP-Bedingungen für ein Projekt einrichten</span><span class="sxs-lookup"><span data-stu-id="4f453-121">Set up PWP terms for a project</span></span>

<span data-ttu-id="4f453-122">Wenn Sie PWP-Bedingungen für ein Projekt einrichten, müssen Sie als Prozentsatz den Mindestbetrag angeben, den ein Kunde für das Projekt bezahlen muss, bevor Sie den Kreditor bezahlen.</span><span class="sxs-lookup"><span data-stu-id="4f453-122">When you set up PWP terms for a project, you must specify, as a percentage, the minimum amount that a customer must pay you for the project before you will pay the vendor.</span></span> <span data-ttu-id="4f453-123">Der Schwellenbetrag wird automatisch auf den Kundenrechnungen für das Projekt berechnet.</span><span class="sxs-lookup"><span data-stu-id="4f453-123">The threshold amount is automatically calculated on the customer invoices for the project.</span></span> <span data-ttu-id="4f453-124">Befolgen Sie diese Schritte, um den Schwellenprozentsatz für PWP-Bedingungen festzulegen.</span><span class="sxs-lookup"><span data-stu-id="4f453-124">Follow these steps to set up the threshold percentage for PWP terms.</span></span>

2. <span data-ttu-id="4f453-126">Suchen und öffnen Sie das Projekt, für das Sie PWP-Bedingungen einrichten möchten.</span><span class="sxs-lookup"><span data-stu-id="4f453-126">Find and open the project that you want to set up PWP terms for.</span></span>
3. <span data-ttu-id="4f453-127">Wählen Sie im Inforegister **Kreditorenvereinbarungen** die Option **Zeile hinzufügen** aus.</span><span class="sxs-lookup"><span data-stu-id="4f453-127">On the **Vendor agreements** FastTab, select **Add line**.</span></span>
3. <span data-ttu-id="4f453-128">Wählen Sie im Feld **Kontocode** eine der folgenden Optionen aus:</span><span class="sxs-lookup"><span data-stu-id="4f453-128">In the **Account code** field, select one of the following options:</span></span>

    - <span data-ttu-id="4f453-129">**Tabelle** – Die PWP-Bedingungen gelten für einen einzelnen Kreditor.</span><span class="sxs-lookup"><span data-stu-id="4f453-129">**Table** – The PWP terms apply to a single vendor.</span></span>
    - <span data-ttu-id="4f453-130">**Gruppe** – Die PWP-Bedingungen gelten für alle Kreditoren in einer Kreditorengruppe.</span><span class="sxs-lookup"><span data-stu-id="4f453-130">**Group** – The PWP terms apply to all vendors in a vendor group.</span></span>
    - <span data-ttu-id="4f453-131">**Alle** – Die PWP-Bedingungen gelten für alle Kreditoren.</span><span class="sxs-lookup"><span data-stu-id="4f453-131">**All** – The PWP terms apply to all vendors.</span></span>

4. <span data-ttu-id="4f453-132">Wenn Sie **Tabelle** oder **Gruppe** im vorherigen Schritt im Feld **Kreditor/Kreditorengruppe** ausgewählt haben, wählen Sie den Kreditor oder die Kreditorengruppe aus, auf die die PWP-Bedingungen zutreffen.</span><span class="sxs-lookup"><span data-stu-id="4f453-132">If you selected **Table** or **Group** in the previous step, in the **Vendor/Vendor group** field, select the vendor or vendor group that the PWP terms apply to.</span></span> <span data-ttu-id="4f453-133">Wenn Sie **Alle** im vorherigen Schritt ausgewählt haben, kann das Feld **Kreditor/Kreditorengruppe** nicht bearbeitet werden.</span><span class="sxs-lookup"><span data-stu-id="4f453-133">If you selected **All** in the previous step, the **Vendor/Vendor group** field can't be edited.</span></span>
5. <span data-ttu-id="4f453-134">Wenn für den Kreditor im Projekt Bedingungen für die Einbehaltung eingerichtet sind, wählen Sie im Feld **Bedingungen für einbehaltene Kreditorenbeträge** die Regel-ID für die Bedingungen zur Einbehaltung aus.</span><span class="sxs-lookup"><span data-stu-id="4f453-134">If vendor retention terms are set up for the vendor in the project, in the **Vendor retention terms** field, select the rule ID for the retention terms.</span></span>
