<span data-ttu-id="a6385-105">Eine projektbasierte Vertragszeile hat *eingeschlossene* Komponenten und *fakturierbare* Komponenten.</span><span class="sxs-lookup"><span data-stu-id="a6385-105">A project-based contract line has *included* components and *chargeable* components.</span></span>

  - <span data-ttu-id="a6385-113">Nicht fakturierbar</span><span class="sxs-lookup"><span data-stu-id="a6385-113">Non-chargeable</span></span>

<span data-ttu-id="a6385-114">Fakturierbare Komponenten können für die Kategorien Aufgaben, Rollen und Transaktionen definiert werden.</span><span class="sxs-lookup"><span data-stu-id="a6385-114">Chargeable components can be defined on tasks, roles, and transaction categories.</span></span>

### <a name="update-a-project-task-as-chargeable-or-non-chargeable"></a><span data-ttu-id="a6385-121">Eine Projektaufgabe als fakturierbar oder nicht fakturierbar aktualisieren</span><span class="sxs-lookup"><span data-stu-id="a6385-121">Update a project task as chargeable or non-chargeable</span></span>
