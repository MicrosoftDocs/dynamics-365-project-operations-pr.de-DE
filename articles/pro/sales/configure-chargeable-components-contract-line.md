# <a name="configure-chargeable-components-of-a-project-based-contract-line---lite"></a><span data-ttu-id="0bd2f-103">Kostenpflichtige Komponenten einer projektbasierten Vertragszeile konfigurieren – Lite</span><span class="sxs-lookup"><span data-stu-id="0bd2f-103">Configure chargeable components of a project-based contract line - lite</span></span>

<span data-ttu-id="0bd2f-105">Eine projektbasierte Vertragszeile hat *eingeschlossene* Komponenten und *fakturierbare* Komponenten.</span><span class="sxs-lookup"><span data-stu-id="0bd2f-105">A project-based contract line has *included* components and *chargeable* components.</span></span>

<span data-ttu-id="0bd2f-106">Eingeschlossene Komponenten sind Komponenten, die Folgendem unterliegen:</span><span class="sxs-lookup"><span data-stu-id="0bd2f-106">Included components are components that are subject to:</span></span>

  - <span data-ttu-id="0bd2f-107">Fakturierungsmethode und Kundenaufteilung</span><span class="sxs-lookup"><span data-stu-id="0bd2f-107">Billing method and customer splits</span></span>

  - <span data-ttu-id="0bd2f-109">In der projektbasierten Vertragszeile definierte Einstellungen für die Rechnungshäufigkeit</span><span class="sxs-lookup"><span data-stu-id="0bd2f-109">Invoice frequency settings defined on the project-based contract line</span></span>

<span data-ttu-id="0bd2f-110">Eine Teilmenge der eingeschlossenen Komponenten kann mit dem Feld **Fakturierungstyp** als fakturierbar markiert werden.</span><span class="sxs-lookup"><span data-stu-id="0bd2f-110">A subset of the included components can be marked as chargeable using the **Billing Type** field.</span></span> <span data-ttu-id="0bd2f-111">Das Feld **Fakturierungstyp** ist ein Optionssatz, der die folgenden Werte im Kontext einer Vertragszeile zulässt:</span><span class="sxs-lookup"><span data-stu-id="0bd2f-111">The **Billing Type** field is an option-set that allows the following values in the context of a contract line:</span></span>

  - <span data-ttu-id="0bd2f-112">Fakturierbar</span><span class="sxs-lookup"><span data-stu-id="0bd2f-112">Chargeable</span></span>
  - <span data-ttu-id="0bd2f-113">Nicht fakturierbar</span><span class="sxs-lookup"><span data-stu-id="0bd2f-113">Non-chargeable</span></span>

<span data-ttu-id="0bd2f-114">Fakturierbare Komponenten können für die Kategorien Aufgaben, Rollen und Transaktionen definiert werden.</span><span class="sxs-lookup"><span data-stu-id="0bd2f-114">Chargeable components can be defined on tasks, roles, and transaction categories.</span></span>

<span data-ttu-id="0bd2f-119">Die für Transaktionskategorien für eine Projektvertragszeile festgelegte Fakturierbarkeit gilt nur für die Transaktionsklasse **Ausgaben**.</span><span class="sxs-lookup"><span data-stu-id="0bd2f-119">Chargeability defined on transaction categories for a project contract line only applies to the **Expense** transaction class.</span></span> <span data-ttu-id="0bd2f-120">Wenn das Feld **Ausgaben einschließen** in einer Vertragszeile auf **Nein** festgelegt wurde, ist die Registerkarte **Fakturierbare Kategorien** nicht verfügbar.</span><span class="sxs-lookup"><span data-stu-id="0bd2f-120">If the **Include Expenses** field is set to **No**, the **Chargeable Categories** tab will not be available.</span></span>

### <a name="update-a-project-task-as-chargeable-or-non-chargeable"></a><span data-ttu-id="0bd2f-121">Eine Projektaufgabe als fakturierbar oder nicht fakturierbar aktualisieren</span><span class="sxs-lookup"><span data-stu-id="0bd2f-121">Update a project task as chargeable or non-chargeable</span></span>
