# <a name="configure-chargeable-components-of-a-project-based-contract-line"></a><span data-ttu-id="7c133-103">Fakturierbare Komponenten einer projektbasierten Vertragszeile konfigurieren</span><span class="sxs-lookup"><span data-stu-id="7c133-103">Configure chargeable components of a project-based contract line</span></span>

<span data-ttu-id="7c133-104">_**Gilt für:** Project Operations für Szenarien basierend auf vorrätigen/nicht-vorrätigen Ressourcen_</span><span class="sxs-lookup"><span data-stu-id="7c133-104">_**Applies To:** Project Operations for resource/non-stocked based scenarios_</span></span>

<span data-ttu-id="7c133-105">Eine projektbasierte Vertragszeile umfasst das Konzept eingeschlossener, fakturierbarer und nicht fakturierbarer Komponenten.</span><span class="sxs-lookup"><span data-stu-id="7c133-105">A project-based contract line has the concept of included, chargeable, and non-chargeable components.</span></span>

<span data-ttu-id="7c133-106">Eingeschlossene Komponenten unterliegen der Fakturierungsmethode, Kundenaufteilungen, nicht zu überschreitenden Grenzwerten und Einstellungen zur Rechnungshäufigkeit, die in der projektbasierten Vertragszeile definiert sind.</span><span class="sxs-lookup"><span data-stu-id="7c133-106">Included components are subject to the billing method, customer splits, not-to-exceed limits, and invoice frequency settings defined on the project-based contract line.</span></span>

<span data-ttu-id="7c133-107">Eine Teilmenge der eingeschlossenen Komponenten kann durch Aktualisierung des Felds **Fakturierungstyp** als fakturierbar markiert werden.</span><span class="sxs-lookup"><span data-stu-id="7c133-107">A subset of the included components can be marked as chargeable by updating the **Billing Type** field.</span></span>

<span data-ttu-id="7c133-108">Fakturierbare Komponenten können für die Kategorien Rollen und Transaktionen definiert werden.</span><span class="sxs-lookup"><span data-stu-id="7c133-108">Chargeable components can be defined on roles, and transaction categories.</span></span>

<span data-ttu-id="7c133-109">Die für Rollen für eine Projektvertragszeile festgelegte Fakturierbarkeit gilt nur für die Transaktionsklasse **Zeit**.</span><span class="sxs-lookup"><span data-stu-id="7c133-109">For a project contract line, the chargeability defined on a role only applies to the **Time** transaction class.</span></span> <span data-ttu-id="7c133-110">Wenn das Feld **Zeit einschließen** in einer Projektvertragszeile auf **Nein** festgelegt wurde, ist die Registerkarte **Fakturierbare Rollen** nicht verfügbar.</span><span class="sxs-lookup"><span data-stu-id="7c133-110">If **Include Time** is set to **No** on a project contract line, the **Chargeable roles** tab isn't available.</span></span>

<span data-ttu-id="7c133-111">Die für Transaktionskategorien für eine Projektvertragszeile festgelegte Fakturierbarkeit gilt nur für die Transaktionsklasse **Ausgaben**.</span><span class="sxs-lookup"><span data-stu-id="7c133-111">Chargeability defined on transaction categories for a project contract line only applies to the **Expense** transaction class.</span></span> <span data-ttu-id="7c133-112">Wenn das Feld **Ausgaben einschließen** in einer Projektvertragszeile auf **Nein** festgelegt wurde, ist die Registerkarte **Fakturierbare Kategorien** nicht verfügbar.</span><span class="sxs-lookup"><span data-stu-id="7c133-112">If **Include Expenses** is set to **No** on a project contract line, the **Chargeable Categories** tab isn't available.</span></span>

<span data-ttu-id="7c133-115">Aktualisieren Sie auf der Registerkarte **Fakturierbare Rollen** einer projektbasierten Vertragszeile im Unterraster **Fakturierbare Kategorien** im **Fakturierungstyp**-Feld den Fakturierungstyp für eine Rolle.</span><span class="sxs-lookup"><span data-stu-id="7c133-115">On the **Chargeable roles** tab of a projec-based contract line, on the **Chargeable Categories** subgrid, in the **Billing Type** field, update the billing type for a role.</span></span>

### <a name="update-a-transaction-category-to-be-chargeable-or-non-chargeable"></a><span data-ttu-id="7c133-116">Eine Transaktionskategorie als fakturierbar oder nicht fakturierbar aktualisieren</span><span class="sxs-lookup"><span data-stu-id="7c133-116">Update a transaction category to be chargeable or non-chargeable</span></span>

<span data-ttu-id="7c133-117">Eine Transaktionskategorie kann für eine bestimmte projektbasierte Vertragszeile fakturierbar oder nicht fakturierbar sein.</span><span class="sxs-lookup"><span data-stu-id="7c133-117">A transaction category can be chargeable or non-chargeable on a specific project-based contract line.</span></span>

<span data-ttu-id="7c133-118">Aktualisieren Sie auf der Registerkarte **Fakturierbare Kategorien** einer projektbasierten Vertragszeile im Unterraster **Fakturierbare Kategorien** im **Fakturierungstyp**-Feld den Fakturierungstyp für eine Transaktion.</span><span class="sxs-lookup"><span data-stu-id="7c133-118">On the **Chargeable Categories** tab of a project-based contract line, on the **Chargeable Categories** subgrid, in the **Billing Type** field, update the billing type for a transaction.</span></span>
