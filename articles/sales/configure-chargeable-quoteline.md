# <a name="configure-the-chargeable-components-of-a-project-based-quote-line"></a><span data-ttu-id="13018-103">Die kostenpflichtigen Komponenten einer projektbasierten Angebotsposition konfigurieren</span><span class="sxs-lookup"><span data-stu-id="13018-103">Configure the chargeable components of a project-based quote line</span></span>

<span data-ttu-id="13018-104">_**Gilt für:** Project Operations für Szenarien basierend auf vorrätigen/nicht-vorrätigen Ressourcen_</span><span class="sxs-lookup"><span data-stu-id="13018-104">_**Applies To:** Project Operations for resource/non-stocked based scenarios_</span></span>

<span data-ttu-id="13018-105">Eine projektbasierte Angebotszeile kann Komponenten und kostenpflichtige Komponenten enthalten.</span><span class="sxs-lookup"><span data-stu-id="13018-105">A project-based quote line can have included components and chargeable components.</span></span>

<span data-ttu-id="13018-106">Enthaltene Komponenten unterliegen der Abrechnungsmethode, Kundensplits, nicht zu überschreitenden Grenzwerten und Rechnungshäufigkeitseinstellungen, die in der projektbasierten Angebotszeile definiert sind.</span><span class="sxs-lookup"><span data-stu-id="13018-106">Included components are subject to the billing method, customer splits, not-to-exceed limits, and invoice frequency settings defined on the project-based quote line.</span></span>

   - <span data-ttu-id="13018-109">Fakturierbar</span><span class="sxs-lookup"><span data-stu-id="13018-109">Chargeable</span></span>
   - <span data-ttu-id="13018-110">Nicht fakturierbar</span><span class="sxs-lookup"><span data-stu-id="13018-110">Non-chargeable</span></span>

<span data-ttu-id="13018-111">Fakturierbare Komponenten können für die Kategorien Rollen und Transaktionen definiert werden.</span><span class="sxs-lookup"><span data-stu-id="13018-111">Chargeable components can be defined on roles and transaction categories.</span></span>

<span data-ttu-id="13018-112">Die für Rollen für eine Projektangebotszeile definierte Aufladbarkeit gilt nur für die **Zeit**-Transaktionsklasse.</span><span class="sxs-lookup"><span data-stu-id="13018-112">Chargeability that is defined on roles for a project quote line only applies to the **Time** transaction class.</span></span> <span data-ttu-id="13018-113">Wenn eine Projektangebotszeile **Zeit einschließen** = **Nein** festgelegt hat, ist die **Fakturierbare Rollen**-Registerkarte nicht verfügbar.</span><span class="sxs-lookup"><span data-stu-id="13018-113">If a project quote line has **Include Time** = **No**, the **Chargeable Roles** tab isn't available.</span></span>

<span data-ttu-id="13018-114">Die für Transaktionskategorien für eine Projektangebotszeile definierte Aufladbarkeit gilt nur für die **Ausgaben**-Transaktionsklasse.</span><span class="sxs-lookup"><span data-stu-id="13018-114">Chargeability that is defined on transaction categories for a project quote line only applies to the **Expense** transaction class.</span></span> <span data-ttu-id="13018-115">Wenn eine Projektangebotszeile **Ausgaben einschließen** = **Nein** festgelegt hat, ist die **Fakturierbare Kategorien**-Registerkarte nicht verfügbar.</span><span class="sxs-lookup"><span data-stu-id="13018-115">If a project quote line has **Include Expenses** = **No**, the **Chargeable Categories** tab isn't available.</span></span>
