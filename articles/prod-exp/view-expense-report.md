# <a name="view-an-expense-report-from-general-ledger-or-procurement-and-sourcing"></a><span data-ttu-id="09ee9-103">Eine Spesenabrechnung aus der Finanzbuchhaltung oder aus Beschaffung und Einkauf anzeigen</span><span class="sxs-lookup"><span data-stu-id="09ee9-103">View an expense report from General ledger or Procurement and sourcing</span></span>

[!include [banner](../includes/banner.md)]

<span data-ttu-id="09ee9-104">Wenn Sie eine Abfrage ausführen, um Transaktionen in der Finanzbuchhaltung oder in Beschaffung und Einkauf anzuzeigen, gibt die Abfrage möglicherweise zu viele Informationen zurück.</span><span class="sxs-lookup"><span data-stu-id="09ee9-104">When you run a query to view transactions in General ledger or Procurement and sourcing, the query might return too much information.</span></span> <span data-ttu-id="09ee9-105">Daher möchten Sie möglicherweise Ihre Suche eingrenzen.</span><span class="sxs-lookup"><span data-stu-id="09ee9-105">Therefore, you might want to narrow your search.</span></span> <span data-ttu-id="09ee9-106">Um die benötigten Informationen zu erhalten, können Sie die Detailinformationen einer Transaktion aufrufen, um die Spesenabrechnung zu sehen, in der sie angezeigt wurde.</span><span class="sxs-lookup"><span data-stu-id="09ee9-106">To get the information that you require, you can drill down into a transaction to see the expense report that it appeared on.</span></span>

<span data-ttu-id="09ee9-107">Für Spesenabrechnungen können Sie die Detailinformationen aus einer Transaktion zum Originalbericht anzeigen, der in der Spesenverwaltung gespeichert ist.</span><span class="sxs-lookup"><span data-stu-id="09ee9-107">For expense reports, you can drill down from a transaction to the original report that is stored in Expense management.</span></span> <span data-ttu-id="09ee9-108">Zusätzlich zum Gesamtbetrag der Ausgaben können Sie die aufgeschlüsselten Transaktionen, die Transaktionen, für die die Mehrwertsteuer erstattet werden kann, und die den Ausgaben beigefügten Belege anzeigen.</span><span class="sxs-lookup"><span data-stu-id="09ee9-108">In addition to the total amount of expenses, you can view the itemized transactions, the transactions that are eligible for recovery of value-added tax (VAT), and the receipts that are attached to the expenses.</span></span>

<span data-ttu-id="09ee9-109">Um Detailinformationen zur ursprünglichen Spesenabrechnung anzuzeigen, in der eine Transaktion vorhanden ist, wählen Sie die Transaktion und dann **Originaldokument** aus.</span><span class="sxs-lookup"><span data-stu-id="09ee9-109">To drill down to the original expense report that a transaction appeared on, select the transaction, and then select **Original document**.</span></span>

<span data-ttu-id="09ee9-110">Wenn Sie einzelne Transaktionen anzeigen, können Sie auch alle zusätzlichen Daten anzeigen, die an eine Transaktion angehängt sind.</span><span class="sxs-lookup"><span data-stu-id="09ee9-110">When you view individual transactions, you can also view any additional data that is attached to a transaction.</span></span> <span data-ttu-id="09ee9-111">Beispielsweise können Sie für eine aufgeschlüsselte Hotelausgabe eine detaillierte Liste der Ausgaben anzeigen, z. B. Internetzugang, Telefonanrufe und Zimmerservice.</span><span class="sxs-lookup"><span data-stu-id="09ee9-111">For example, for an itemized hotel expense, you can view a detailed list of expenses such as internet access, telephone calls, and room service.</span></span>
